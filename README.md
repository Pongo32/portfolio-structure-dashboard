# Портфельная структура для инвестиций

## Локальная сборка без CDN

### Установка зависимостей

```bash
npm install
```

### Настройка переменных окружения

1. Скопируйте `.env.example` в `.env`
2. Заполните ваши API ключи

### Разработка

```bash
npm run dev
```

Или используйте Windows batch файл:
```bash
start-local.bat
```

### Сборка для продакшн

```bash
npm run build
```

### Предварительный просмотр сборки

```bash
npm run preview
```
